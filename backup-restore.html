⚠ Important: This tool must be opened from the same location as index.html — same folder or same server — so it shares the same localStorage origin. Opening it from a different location will see empty storage.

Current Storage

Click Refresh to scan storage.

Step 1 — Backup

Exports all app data as a single .json file. Store it somewhere safe before making changes.

Step 2 — Clear (optional)

Wipes all app data from localStorage. Only do this after a successful backup. Agent name and station are preserved so the form still works after clearing.

Step 2b — Factory Defaults

Seeds the built-in Default Aircraft Ref and Initial Assets tables directly to localStorage. Use this to restore these reference tables to their factory state, or to ensure they are captured in your next backup even if they have never been modified.

Step 3 — Restore

Select a backup .json file to restore. Each key is validated before writing. Existing data for the same key is overwritten.
